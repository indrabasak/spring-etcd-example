Spring etcd Example
==================================
This is an example of using [etcd v3.0](https://coreos.com/etcd/) with Spring 
Boot. The application uses [jetcd v0.0.1](https://github.com/coreos/jetcd), 
a Java client for etcd, to communicate with the database.
